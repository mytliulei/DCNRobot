ActiveState ActiveTcl 8.4.19.2.291226                                              Sep 21, 2009
_____________________________________________________________________________

Welcome to ActiveTcl, ActiveState's free binary distribution of Tcl
and major extensions, available for Linux, Solaris and Windows.

ActiveState is committed to making Tcl easy to install and use on all
major platforms. This second release of ActiveTcl includes 
the most stable versions of major extensions in binary form.

We appreciate your suggestions regarding add-on packages for 
ActiveTcl. Please send your comments to to 
ActiveTcl-feedback@ActiveState.com.

_____________________________________________________________________________

Installing ActiveTcl 8.4.19.2
_____________________________________________________________________________

To install ActiveTcl 8.4.19.2, run the "install.sh" script found in the
same directory as this document.

To uninstall ActiveTcl, run the "uninstall" application found in the 
directory "lib/ppm/log/activetcl8.4".
